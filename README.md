# JavaOMS
JavaOMS: the order managament system implemented by Java

## Services Overview

- Product Service
- Order Service
- Inventory Service
- Notification Service
- API Gateway using Spring Cloud Gateway MVC

## Tech Stack

The technologies used in this project are:

- Spring Boot
- Mongo DB
- MySQL
- Kafka
